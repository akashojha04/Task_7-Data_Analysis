# Sales Data Analysis & Visualization using SQLite + Python
--

# This project demonstrates end-to-end data analysis using:
SQLite Database
Python (Jupyter Notebook)
Pandas for data manipulation
Matplotlib for visualization

--

# The objective of this project is to:
*Store structured sales data in a database*
*Perform SQL aggregation queries*
*Analyze sales performance*
*Create visual insights using Python visualizations*
*This project simulates a real-world business sales reporting workflow.*

--

# Tools & Technologies Used
**SQLite**
* Lightweight relational database
* Used for storing sales transactions
* Supports SQL queries with aggregation

**Python**
* Core programming language for analysis

**Pandas**
* Loading SQL data into DataFrame
* Data cleaning
* Grouping & aggregation
* Date transformation
* Feature engineering

**Matplotlib**
Bar charts
Line charts
Pie charts
Business reporting visuals
